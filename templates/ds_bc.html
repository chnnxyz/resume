Contenidos del Curso

El curso comprende los siguientes temas:

  • Manejo de terminal UNIX-like
  • Git y Github
  • Básicos de Python: Variables y estructuras de control
  • Manejo de datos con Pandas, Arreglos con Numpy, visualización con MatPlotLib y Seaborn
  • Limpieza y preprocesamiento de datos
  • Modelos sencillos de clasificación y regresión
  • Clustering y algoritmos de recomendación
  • Redes neuronales artificiales
  • Computer Vision (redes neuronales convolucionales)
  • Docker
  • AWS S3, Lambda, EC2 y RDS
  • Big Data con PySpark
